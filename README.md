# Sublime Typescript Transpiler
A sublime package to transpile typescript with *ctrl-s* action.

## Structure to work
This package is intended to use with ts and js folder below root folder.
```
.
├── index.html
├── js
│   ├── std.js
│   ├── test.js
│   └── testSuit.js
├── std.js
└── ts
    ├── std.ts
    ├── test.ts
    ├── testSuit.ts
    └── tsconfig.json
```
